============================================================
       業務進捗管理システム [ 進捗くん ＆ 司令塔 ]
                  Final Specification
============================================================

【1. プロジェクト概要】
------------------------------------------------------------
本システムは、小規模プロジェクトにおける進捗管理のコストを
最小化するために開発された「自律稼働型」ツール群である。
開発者の手を離れ、事務方だけで運用が完結することを目標とする。

【2. 設計思想（魂の三原則）】
------------------------------------------------------------
1. [SSOT] 共有フォルダのconfig.jsonが唯一の正解である。
2. [透明性] ログはCSV。万が一の際もExcelで救出できる。
3. [脱・属人化] 開発者がいなくても現場が回る仕組みを作る。

【3. 構成モジュール】
------------------------------------------------------------
■1. 利用者アプリ：『進捗入力くん』 (user_app.py)
   - リアルタイム経過タイマー表示 (hh:mm:ss)
   - 前タスクの自動終了を伴う「ワンクリック切り替え」
   - 集中を妨げない「最前面ピン留め」トグル機能
   - Excelでそのまま開ける「BOM付きUTF-8」出力

■2. 管理者アプリ：『進捗管理者パネル』 (admin_app.py)
   - 現場の動きを一望できる「1秒更新ライブモニター」
   - タスク追加・名前割当をGUIで完結（JSON編集不要）
   - 全員分のログを一括統合する「エクスポート機能」
   - 項目増に対応したスクロール可能な設定画面

【4. 標準運用フロー】
------------------------------------------------------------
[STEP 1] 利用者がexeを起動し、共有フォルダを設定。
[STEP 2] 管理者がパネルを開き、検出されたPC名に名前を割当。
[STEP 3] 利用者が再起動。自分専用の環境で入力開始。
[STEP 4] 月末、管理者がボタン一つで全データを集計。

【5. 技術スタック】
------------------------------------------------------------
  Language : Python 3.12
  UI       : CustomTkinter
  Storage  : JSON / CSV (utf-8-sig)
  Build    : PyInstaller (--onefile --noconsole)
